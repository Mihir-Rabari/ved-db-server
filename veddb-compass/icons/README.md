# VedDB Compass Icons

This directory contains application icons for VedDB Compass.

## Required Icons

- `32x32.png` - 32x32 pixel PNG icon
- `128x128.png` - 128x128 pixel PNG icon
- `128x128@2x.png` - 256x256 pixel PNG icon (2x retina)
- `icon.icns` - macOS icon file
- `icon.ico` - Windows icon file

## Generating Icons

Icons will be generated from a source SVG or PNG file using appropriate tools:

- For macOS (.icns): Use `iconutil` or online converters
- For Windows (.ico): Use ImageMagick or online converters
- For PNG: Use image editing software
