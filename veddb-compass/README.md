# VedDB Compass

Desktop GUI application for VedDB database management, built with Tauri + React + TypeScript.

## Development

```bash
# Install dependencies
npm install

# Run in development mode
npm run tauri dev

# Build for production
npm run tauri build
```

## Features (To be implemented in Task 17)

- Connection management
- Database explorer
- Query builder
- Document viewer and editor
- Aggregation pipeline builder
- Real-time metrics dashboard
- Index management
- User management
- Import/export functionality
- Dark/light theme support

## Tech Stack

- **Frontend**: React 18 + TypeScript + Vite
- **Backend**: Rust + Tauri
- **UI Components**: Will use shadcn/ui + Tailwind CSS
- **State Management**: Zustand
- **Charts**: Recharts
- **Code Editor**: Monaco Editor
